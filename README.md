## 1) OOP C++ Assignment — Card Game

**Path:** `OOP_Cpp_Assignments/A1_project`

Terminal-based, turn-driven card game implemented in C++.

### Note
This project was completed as part of a university assignment.
Some infrastructure files (starter framework / utility scaffolding) were provided by tutors.
My contribution focuses on implementing the game logic, class interactions, command handling, and memory management.

### Highlights
- Implemented class hierarchy for cards, entities, board, and game loop
- Applied inheritance and polymorphism (`Card`, `ActionCard`, `OceanCard`, etc.)
- Built command parsing and execution flow (`swim`, `float`, `action`, `deck`, `hand`, `board`, `quit`)
- Implemented dynamic memory cleanup via destructors
- Practiced copy-constructor and deep-copy patterns in separate exercises

### Core Concepts
- OOP: abstract classes, virtual methods, `override`, virtual destructors
- STL containers: `std::vector`, `std::array`, `std::optional`, `std::string`
- Pointer ownership and resource lifetime
- CLI architecture and state management

### Build & Run
```bash
cd OOP_Cpp_Assignments/A1_project
make
./a1 <PLAYER_COUNT> <CONFIG_PATH>
```
